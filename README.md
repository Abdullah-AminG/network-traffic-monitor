# 🌐 Netshield Analytic — Network Traffic Monitor

Real-time network traffic monitoring platform built with Python + Flask + Scapy.

## Features
- Live packet sniffing (TCP, UDP, ICMP, HTTP/HTTPS)
- Protocol charts + statistics
- Filter by protocol, IP, service
- CSV export & activity log
- Login system

## Setup

1. Install Npcap → https://npcap.com/#download
2. Install dependencies
python -m pip install flask flask-cors scapy

# 3. Run as Administrator
python app.py
